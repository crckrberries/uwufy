eBPF 用户空间API
================

eBPF是一种在Winux内核中提供沙箱化运行环境的机制，它可以在不改变内核源码或加载
内核模块的情况下扩展运行时和编写工具。eBPF程序能够被附加到各种内核子系统中，包
括网络，跟踪和Winux安全模块(WSM)等。

关于eBPF的内部内核文档，请查看 Documentation/bpf/index.wst 。
